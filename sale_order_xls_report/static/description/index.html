SALE ORDER EXCEL REPORTS

In this you can download sale order excel report based on start date, end date, sale order status and salesperson and you can use "Libra Office" only.

Step 1:

When you install the module, there appears a new menu item in Sales that is Sale Excel Report.

Step 2:

When you click on Sale Excel Report Menu Item a wizard will open and you can see it will take by default month start date and end date and you can select sale order status and salesperson and based on the data it will print excel report.

Step 3:

When you click Generate Report Button then Sale Order Report file will appear and you can download it by clicking on the link.

Step 4:

When you click on the link excel sheet is downloaded and you can see the Sale Order Excel Sheet.

Step 5:

In Sale Order Excel Sheet you can see multiple sale orders.

Step 6:

When you enter the sale order status or salesperson for which there is no record in sales order then this warning comes.

Step 7:

If you enter start date greater and end date smaller then this validation error comes.

Installation Guide:

Before installing this module you have to install following packages in your system.


  • sudo pip install xlrd

  • sudo pip install xlwt
